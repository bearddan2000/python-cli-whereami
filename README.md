# python-cli-whereami

## Description
This is a POC project to demonstrate
a python console app.

## Tech stack
- perl
- python3
  - os

## Docker stack
- docker-cli
- alpine:latest
- centos:latest
- ubuntu:latest

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
```sudo ./install.sh -u```

## To stop (optional)
```sudo ./install.sh -d```
